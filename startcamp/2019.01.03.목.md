```python
text = input("입력해주세요:")

if text.lower()=="menu" or text.lower()=="메뉴":
    print("밥먹고싶다")
else:
    print("밥없음")
```

이렇게하면 잘 나온다 ㅎㅎㅎㅎㅎ

틀린이유는!! "메뉴"는 항상 True이기 때문이다.

왜 항상 True냐면, 음.... 그 어제배운건데! 공백이나 0 이런거는 항상 False고,

나머지는 항상 True라고 했다!

그래서 위에껄로 하면 뭘 입력해도 "밥먹고싶다"로 나온 것이다.
